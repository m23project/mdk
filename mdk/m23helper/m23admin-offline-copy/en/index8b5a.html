Help: Edit package selections

In this dialog you can edit or delete existing and create new package selections as well as export the selected packages' names into a file or import package names from a file into the list of preselected packages.

Step by step

  1. Click on "Package selection", "Normal packages" or "Special packages (for skilled users only)".

  2. Enter the search criteria or leave the search field empty to list all packages.

  3. Click on "Search".

  4. Choose the desired packages from the "Found packages" list.

  5. Click on "Preselect".

  6. Now you can delete serveral packages from the selection by checking the packages and click on "Discard selected".

  7. Packages can be added by searching in the search dialog and preselecting the desired packages.

  8. Save this selection under the same or a different selection name.

Deletion of selection

On the bottom of the page you can select a selection name and click on "Delete".

Im/Export selected packages

A click on "Export selected packages" stores the packages listed under "Preselected packages" to a file. For the import of packages click on "Browse ..." (and otherwise labeled in your browser respectively) and choose the file with the preselected packages. Then click on "Upload".

Hint

The package names are seperated by blanks in the exported files.
