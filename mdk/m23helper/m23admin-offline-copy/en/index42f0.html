Help: Server status

The shown status of your m23 should be used for copying and pasting into the m23 forum. It is presented in the syntax of the forum software and contains e.g. the m23 version number and the result of a network connection check. The copying of these status information allows a fast answer of your question, because lots of basic questions doen't need to be asked and answered. Please add this status log when you start a new thread (or asked to do so) in the forum.
