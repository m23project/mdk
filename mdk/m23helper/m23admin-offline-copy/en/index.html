What is m23?

m23 is the software distribution and client management system for Debian Linux and some Ubuntu versions.

With m23 you can install and manage hundreds of clients over an existing network. m23 can partition and format clients and install a Linux base system and thousands of applications. All this you can do from one place with the m23 administration GUI that you can use with any web browser in your intranet.

m23 license

m23 is licensed under the GPL (General Public License), that grants you to use m23 without paying for it. You can modify m23 and distribute your changes, but you always have to give away your source code and put your changes unter the GPL again. So m23 and all installable applications will never get you into license trouble. Now you have the unique possibility to make your IT free of proprietary and expensive software solutions.

m23 community

You can get the m23 community support on voluntary basis for free. "Voluntary" means that you don't have the right to get your problems with m23 solved under all circumstances. The m23 community support covers problems with the installation of m23 in standard configurations and the fixing of bugs in m23 only.

You need to have good knowlegde of Debian and you have to be willing to give accurate fault tracing and to solve the problem together with the m23 developer, if you want to engage the m23 community support.

If you need more help or other services for m23, you should use m23's commercial support.

At http://m23.sf.net you can find:

  • User forum

  • Documentation

  • ISO images for burning m23 server installation CDs

  • SourceCodes of all m23 components

  • etc.

Commercial support for m23 and other OSS solutions

The commercial support (German only) includes services for m23 and other OpenSource related topics:

  • (Online) m23 trainings for developers and administrators

  • Development of individual m23 extensions and customisations

  • Consulting: Planning of clients, servers and networks to use with m23 and general OpenSource consulting

  • Creation of virtual machines and virtual appliances

  • Trainings for Linux, OpenSource, script programming with BASH, awk, sed, grep and co. as well as schoolings for HTML, PHP and MySQL

Additional information about the services can be found at www.goos-habermann.de.
