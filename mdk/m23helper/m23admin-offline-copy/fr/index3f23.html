Aide: Gestion des adresses IP

Ce dialogue vous aide à gérer les paramètres de réseau pour des machines hors de m23.

Postes client unitaires / paramètres du réseau

Ici, il vous est possible d'assigner des paramètres réseau spécifiques à des machines individuelles ou bien de réserver des valeurs spécifiques qui ne seront alors plus disponibles pour des postes client m23, ainsi il ne sera plus possible d'assigner à tort ces valeurs à un poste client m23.

  • Réserver le nom de poste client / l' IP / l'adresse MAC: En utilisant la fonction Réserver le nom de poste client / l' IP / l'adresse MAC, vous pouvez réserver un nom de poste client, une adresse IP ou une adresse MAC (par exemple l'adresse IP d'une imprimante réseau), afin que cette valeur ne puisse pas être utilisée à tort par des postes client m23. Entrez l'adresse IP, l'adresse MAC ou le nom que vous souhaitez réserver dans les champs "Nom du poste client", "Adresse IP" ou "Adresse MAC", puis cliquez sur "Réserver le nom de poste client / l' IP / l'adresse MAC". L'utilisation de la fonction "Réserver le nom de poste client / l' IP / l'adresse MAC" nécessite au moins une valeur dans un des champs marqués par 1.

  • Ajouter des paramètres au server DHCP: Si vous souhaitez assigner des paramètres réseau à une machine (externe) en utilisant DHCP, entrez les valeurs désirées dans les champs "Nom du poste client", "Adresse IP", "Masque réseau", "Adresse MAC" et "Passerelle", puis cliquez sur "Ajouter des paramètres au server DHCP". Tous les champs marqués par 2 sont nécessaires à cette fonction.

Interval des adresses IP

Pour les plages d'adresses IP, une fonctionnalité similaire est disponible. Contrairement à la fonction "Postes client unitaires / paramètres du réseau", vous n'y entrerez pas des machines individuelles ou des paramètres de réseau unitaires, mais des plages d'adresses IP qui ne seront plus disponibles pour m23 ou bien dont les adresses IP seront assignées dynamiquement par le serveur DHCP aux machines n'étant pas de postes client m23.

  • Réserver la plage d'adresses IP: Entrez la première et la dernière adresse IP dans les champs "Première adresse IP" et "Dernière adresse IP" et le masque de réseau correspondant ("Masque réseau"), puis cliquez sur "Réserver la plage d'adresses IP", s.v.p. Toutes les adresses IP dans cette plage (la première et la dernière adresse IP incluses) seront ne seront plus disponibles pour les postes client m23. Les valeurs marquées par 1 sont nécessaires pour cette fonction.

  • Ajouter une plage d'adresses IP dynamique: Entrez la première et la dernière adresse IP dans les champs "Première adresse IP" et "Dernière adresse IP" et le masque réseau correspondant ("Masque réseau") ainsi que la passerelle ("Passerelle") puis cliquez sur "Ajouter une plage d'adresses IP dynamique", s.v.p. Toutes les adresses IP dans la plage indiqué (la première et la dernière adresse IP incluses) seront assignées dynamiquement aux machines cherchant un serveur DHCP sur le réseau. Les valeurs marquées avec 2 sont nécessaires pour cette fonction.

Tableaux d'adresses

Le trois tableaux "Adresses IP utilisées", "Plages d'adresses IP dynamiques" et "Plages d'adresses IP réservées" contiennent les valeurs gérées par la gestion d'adresses IP. Le champ "Rechercher" permet d'entrer un critère de recherche (IP, masque de réseau, MAC, nom d'un groupe ou d'autres propriétés d'un poste client) pour restreindre le nombre d valeurs affichées dans le tableau. Pour effacer les paramètres d'une des lignes du tableau, cochez la case en fin de la ligne correspondante, puis cliquez sur "supprimer" ou utilisez le bouton "supprimer" à la fin de la ligne.
