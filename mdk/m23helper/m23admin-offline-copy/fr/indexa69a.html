Aide: Effacer le poste client

Ici, vous pouvez effacer le poste client m23 sélectionné. L'effacement entraîne la suppression du poste client de la base de données, mais le poste client lui-même ne sera pas modifié. Après cet effacement, vous ne pourrez plus administrer le poste client avec m23. Pour accomplir l'effacement, cliquez sur "Effacer".

S'il s'agit d'une machine virtuelle créée par m23, vous pouvez également effacer les fichiers correspondants, après avoir coché la case "Effacer également la machine virtuelle". La machine virtuelle sera définitivement perdue.

Notez:

Si vous souhaitez supprimer toute connection entre m23 et le poste client (effacement des scripts d'initialisation de m23, de la détection du matériel et s'il existe, du proxy pour les paquets APT), exécutez les commandes listées dans "Commandes pour l'effacement des composants m23" en tant qu'utilisateur root sur le poste client (après que ce dernier ait été effacé de la base de données m23).
