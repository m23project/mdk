Aide: Service de téléadministration de m23

Le service de téléadministration de m23 est un service payant fourni par goos-habermann.de, où le support de goos-habermann.de peut avoir accès a votre (vos) serveur(s) m23 afin de vous aider avec vos tâches administratives. Vous trouverez plus d'informations sur le service de téléadministration de m23 et sur nos tarifs et conditions générales sur goos-habermann.de.

Enregistrement

Pour que vous puissiez profiter du service de téléadministration de m23, vous avez besoin d'une clef d'activation. L'enregistrement de cette clef est exempte de frais, mais l'usage du support de goos-habermann.de est payant. Vous pouvez demander votre clef d'activation ici: «Demander un nouveau compte pour le service de téléadministration de m23». Entrez toutes les informations nécessaires dans le formulaire (l'indication dans «Commentaire» est optionelle). Assurez-vous que vos données sont correctes afin que nous puissions traiter votre demande. Si vous souhaitez que goos-habermann.de s'occupe de plusieurs serveurs m23, il vous faudra une clef d'activation distincte pour chaque serveur, par conséquent un enregistrement pour chaque serveur . La clef d'activation doit être demandée depuis l'interface web du serveur m23 correspondant. Toutes les données sauf «Nom ou numéro IP du serveur» (qui est nécessaire pour l'identification de votre serveur par goos-habermann.de) peuvent rester inchangées. Ensuite, cliquez sur «Envoyer à goos-habermann.de» pour envoyer vos données à goos-habermann.de.

Sécurité

Afin de protéger vos données d'un accès non autorisé, elles seront cryptées puis envoyées à goos-habermann.de. C'est pourquoi, goos-habermann.de utilise son propre certificat (auto-signé), devant être importé dans votre navigateur web auparavant. Pour vous rassurer que le certificat a déjà été importé ou pour l'importer, cliquez sur «». Votre navigateur web va vous afficher une alerte, mentionnant que l'éditeur du certificat est inconnu et vous demandant ce que vous souhaitez faire. Acceptez le certificat (sinon, la communication ne pourra être cryptée). Le certificat a les propriétés suivantes:

• valide à partir de: 27.11.2011 (18:16:30)
• valide jusqu'à le 24.11.2021 (18:16:30)
• numéro de série: 4E:D2:70:6E
• editeur: CN = m23-Projekt
• propriétaire: CN = ras.goos-habermann.de, O = m23 project
• empreinte digitale MD5: 4e:89:d6:7e:09:0e:81:3e:99:27:a4:4e:44:48:2e:b0
• empreinte digitale SHA1: 6c:31:47:fe:88:9b:05:f8:20:23:25:49:2b:4f:7e:c4:d8:10:0a:cc

Activation

Après avoir re&ccdil;u votre clef d'activation, vous devez la copier depuis le courriel dans le champ «Clef d'activation du service de téléadministration» du dialogue «Importer la clef d'activation du service de téléadministration de m23». Enfin, cliquez sur «Importer».
