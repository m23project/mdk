Aide: Centre de contrôle

Le centre de contrôle vous permet d'accéder aux informations concernant le poste client et vous offre une grande variété d'outils d'administration.

Remarques sur la virtualisation

m23 comporte la solution de virtualisation VirtualBox-OSE, qui permet d'installer et administrer des postes client virtuels sur des postes client m23 réels (physiques) ou le serveur m23.

Démarrer, suspendre et arrêter un poste client m23 virtuel

Lorsqu'il s'agit d'un poste client m23 ayant été créé par la virtualisation VirtualBox OSE integrée dans m23,

vous pouvez démarrer, suspendre et arrêter le poste client directement depuis l'interface m23.

Vous avez le choix entre deux modes d'opération.

Dans le"mode sans affichage" (dit headless mode) vous ne voyez pas de moniteur virtuel.

Vous démarrez le poste client dans ce mode en cliquant sur "Démarrer la machine virtuelle". Dépendemment des démons installés, vous avez d'autres possibilités pour accéder au système virtuel.

m23 vous met également à disposition un mode graphique, permettant d'avoir accès à la VM dès le démarrage.

En cliquant sur "Démarrer la machine virtuelle avec terminal en mode graphique", la VM démarre et ouvre un serveur VNC, accessible par un client VNC.

Vous trouverez les informations de connexion dans les "Informations sur le poste client" dans la catégorie "Virtualisation" dans les deux champs "Adresse du terminal en mode graphique de la machine virtuelle" et "Mot de passe du terminal en mode graphique de la machine virtuelle".

Dépendant de la vitesse de l'hôte virtuel, l'affichage peut prendre un peu de temps à apparaitre. Le cas échéant, cliquez sur "Actualiser" plusieurs fois. En cliquant sur "Lancer l'applet VNC", vous pouvez ouvrir le "TightVNC Java Viewer" (si Java est installé dans votre navigateur web) et accéder à votre poste client m23 virtuel directement à partir du navigateur web. Comme précedemment, un peu de temps peut être nécessaire pour obtenir l'affichage du poste client m23 . Cliquez sur "Actualiser" le cas échéant.

Pour suspendre une VM, cliquez sur "Mettre en pause la machine virtuelle"; afin d'arrêter complètement la virtualisation, cliquez sur "Arrêter la machine virtuelle".

Établir un hôte de VM

Si vous souhaitez utiliser ce poste client m23 comme hôte pour des postes client m23 virtuels, cliquez sur "Installer l'hôte de machines virtuelles". Le logiciel de virtualisation (s'il est disponible pour la distribution installé) sera installé sur le poste client m23 et sera automatiquement intégré dans la liste des hôtes virtuels. Après une installation concluante, vous pouvez créer et démarrer des postes client m23 virtuels en utilisant la partie du menu "Créer une machine virtuelle".
