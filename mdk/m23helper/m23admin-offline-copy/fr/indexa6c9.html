Aide: Changer l'état du poste client

Ici, vous pouvez changer l'état du poste client. Ceci peut être utile pour le débogage ou pour des autres raisons.

Attention

Ne modifiez cet état qu'en pleine connaissance des conséquences.

Lorsque vous cliquez sur le nom d'un client, des informations détaillées s'afficheront pour le client sélectionné; de plus, vous aurez accès au centre de contrôle.

Signification des couleurs des symboles

La couleur du symbole indique l'état de l'installation du poste client.

  • rouge: Le poste client a été ajouté et la détection du matériel automatique n'est pas encore terminée.

  • jaune: Vous pouvez formater et partitionner le poste client, le système de base et son interface graphique seront installés automatiquement.

  • vert: Le système de base du poste client est installé et vous pouvez y ajouter les logiciels supplémentaires.

  • bleu: Le poste client est en cours d'installation des logiciels supplémentaires.

  • orange: Le poste client est dans un état critique. Cela signifie qu'une erreur s'est produite pendant l'installation; elle doit être résolue par l'administrateur. Le centre de contrôle, vous propose plusieurs solutions afin de résoudre cet état critique.

  • blanc: Poste client modèle pour l'installation de masse à partir duquel les configurations seront transmises aux autres postes client.

  • Bogue: Indique que le poste client se trouve en mode de débogage.

  • gris: Ce client, purement virtuel, permet de sauvegarder les propriétés du client (IP, nom du client, MAC), dans le but qu'elles ne soient pas utilisées accidentellement.

Travaux

Dans cette colonne, s'affiche le nombre des tâches en attente avant la barre oblique et le nombre total des tâches après celle-ci.

Opérer sur plusieurs postes clients

Sélectionnez plusieurs postes client en les cochant dans la liste correspondante.

Vous pouvez lancer les actions sur les postes client choisis:

  • Effacer d'un groupe: Sélectionnez le groupe duquel les postes client seront à retirer. Si un poste client n'appartient pas dans ce groupe, aucune action ne sera excercée sur lui.

  • Ajouter à un groupe: Sélectionnez le groupe auquel seront ajoutés les postes client. Si un poste client est déjà dans ce groupe, son appartenance au groupe restera inchangée.

  • Effacer: Efface les postes client sélectionnés

  • Déplacer dans ce groupe: Cette fonction déplace un poste client dans un groupe différent.

  • Restaurer: Réinstallation complète des postes clients.

  • Comparer l'état de paquets: Comparaison de l'état des paquets entre deux postes clients.

Notez:

Pour actualiser l'état de vos postes client, utilisez la fonction de rechargement de la page dans votre navigateur web (par exemple en appuyant sur la touche F5)

Astuces

En cliquant sur le symbole d'état d'un poste client, vous actualisez son état. Attention, vous ne devriez changer l'état d'un poste client que si vous savez exactement ce que vous faites.

Le mode de débogage peut être (dés)activé en cliquant sur l'icône du bogue.
