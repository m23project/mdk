Aide: Démons et programmes

Démons

Notez

Des démons, sont des programmes serveur toujours actifs en tâche de fond dont on ne visualise l'exécution correcte de leur travaux ;-). Par exemple, le serveur internet Apache et le serveur de base de données MySQL, sont des démons.

Signification des couleurs

  • vert: Le démon est actif en tâche de fond et peut être arrêté ou relancé. Il n'est pas recommandé d'arrêter un démon.

  • jaune: Le démon est installé, non lancé, mais activable. Plusieurs démons pourront être lancés seulement s'il sont ponctuellement nécessaires. Un démon dans cet état peut être lancé.

  • rouge: Le démon n'est pas installé et ne peut être activé. Dans ce cas, un composant important de m23 a été effacé ou n'a pas été installé. Vous pouvez l'installer en choisissant «Installer» après le démon, puis en cliquant sur «Exécuter les actions».

Description des démons

  • apache: Apache est un serveur internet faisant partie du cœur de m23. L'interface d'administration et les travaux à exécuter sur les postes client sont générés via apache par PHP.

  • mysql: MySQL est un serveur de base de données administrant par exemple les propriétés des postes client.

  • dhcpd3: Le serveur DHCP assigne les adresses IP pour l'installation des clients. Il ne peut être lancé que si au moins un poste client doit être installé. L'état «jaune» ne signifie pas une erreur.

  • slapd: est le démon du serveur LDAP OpenLDAP. LDAP permet de stocker les identifiants de connexion sur un serveur central et de les utiliser depuis tout client pouvant accéder à ce serveur.
