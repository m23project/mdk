Aide: Modifier les sources de paquets

Information

Des paquets logiciels peuvent être installés à partir de différentes sources (HTTP,FTP,CDRom, etc ...). Des sources différentes sont nécessaires, lorsque tous les paquets logiciels ne peuvent être installés depuis un même support. Pour vous fournir la possibilité d'administrer simplement ces sources, vous pouvez les déclarer comme requises par la distribution choisie.

  • Charger une source de paquets: Pour le chargement, choisissez la source de paquets dans la liste en haut à droite, puis cliquez sur «Charger», elle sera chargé dans l'éditeur.

  • Effacer une source de paquets: Pour effacer la source de paquets chargé dans l'éditeur, cliquez sur le bouton «Effacer» et confirmez l'effacement au dialogue suivant.

  • Enregistrer une source de paquets: La source de paquets chargée dans l'éditeur sera toujours celle qui sera enregistrée, indépendant de celle sélectionnée dans la liste. Choisissez d'abord la distribution à laquelle la liste des paquets doit s'appliquer. Après avoir choisi la distribution, il est possible que vous deviez entrer des valeurs supplémentaires. De plus, sélectionnez les interfaces graphiques supportés dans la liste des sources de paquets sous le champ de l'éditeur et la/les architecture(s) processeur(s) supportée(s) sous "Architecture du processeur". Trouvez un nom pour la liste de paquets et entrez-le dans le champ de saisie sous «Nom de source de paquets». Si vous avez chargé une source de paquets, le nom sera entrée automatiquement. Si vous voulez, vous pouvez entrer un commentaire dans le champ «Description»; puis, cliquez sur «Enregistrer».

  • Tester une source de paquets: Après avoir entré une source de paquets, vous devriez tester si toutes les sources fonctionnent correctement. Après un clic sur le bouton «Tester», le programme tente de télécharger les descriptions des paquets puis lance un test. Après l'achèvement de ce test un rapport apparaîtra; cela peut prendre quelques minutes jusqu'à son apparission. La vitesse dépend de votre connection internet et de la vitesse des serveurs de sources de paquets utilisés.

  • Entrer le site miroir: Si vous souhaitez utiliser un site miroir différent du site miroir standard est ftp.debian.org pour l'installation du système de base, entrez-le sur une nouvelle ligne dans la source de paquets comme décrit ci-après: #mirror: [URL des paquets]. URL des paquets signifie le protocole (http, ftp, ...), le nom d'hôte ou l'adresse IP et le répertoire, où les paquets peuvent être trouvés. Une ligne valide pourrait se présenter comme cela, par exemple: #mirror: http://192.168.7.14/debianCDs

Information supplémentaire

  • Si vous sauvegardez une source de paquets sous un nom existant, l'ancienne source de paquets sera remplacée.

  • Les interfaces graphiques dépendent directement d'une source de paquets. Vous pouvez uniquement utiliser les interfaces graphiques choisies dans ce dialogue pour un poste client donné. Pour cette raison, vous devriez seulement choisir les interfaces graphiques étant réellement supportées dans les sources de paquets.
