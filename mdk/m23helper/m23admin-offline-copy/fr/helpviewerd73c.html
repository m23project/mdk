Aide: Réstitution de votre serveur m23

Si une panne survenait sur votre serveur m23, vous pouvez restaurer votre serveur dans un des états sauvegardés au préalable en utilisant une des sauvegardes disponibles. Procédez comme suit:

  1. Installez votre serveur m23 sur un disque dur sain avec le CD d'installation du serveur m23. Vous pouvez télécharger l'image ISO sur m23.sf.net pour la graver sur un CD.

  2. Enregistrez la dernière archive de sauvegarde sur l'ordinateur ou se trouve la clé GPG pour le chiffrement et déchiffrement.

  3. S'il s'agit d'un système basé sur Linux, vous pouvez déchiffrer le fichier de sauvegarde en utilisant la ligne de commande ci-dessous:

    gpg -d < m23serverBackup-*.tar.gpg > m23backup.tar

    Sinon, utilisez l'interface graphique GPG de votre système d'exploitation.

  4. Copiez le fichier de sauvegarde déchiffré «m23backup.tar» sur le serveur m23 nouvellement installé dans le répertoire «/tmp». Depuis Linux, vous pouvez le copier par SCP:

    scp m23backup.tar root@[IP]:/tmp

    Remplacez «[IP]» par l'adresse IP de votre serveur m23. Sous Windows, vous pouvez utiliser le programme WinSCP (www.winscp.net).

  5. Entrez le système du serveur m23 comme root.

  6. Placez-vous dans le répertoire «/tmp» par la commande:

    cd /tmp

  7. Créez un nouveau répertoire pour les données déchiffrées et placez-vous dans celui-ci:

    mkdir restore; cd restore

  8. Décompressez le fichier par la commande:

    tar xfv /tmp/m23backup.tar

  9. Démarrez le processus de restauration par la commande:

    /m23/bin/m23Restore

  10. Après la fin de la restauration vous devrez redémarrer votre serveur m23 , p.ex. avec:

    reboot

  11. Maintenant, votre serveur m23 restauré dans l'état de la sauvegarde utilisée devrait être fonctionnel.
