Hilfe: Paketstatus vergleichen

Mit diesem Dialog können Sie den Paketstatus von zwei Clients miteinander vergleichen. Wählen Sie zum Vergleichen unter "1. Client oder Statusdatei" und "2. Client oder Statusdatei" die Datenquellen für die linke und rechte Spalte aus. Sollten Sie diesen Dialog über "Clients-Überblick" aufgerufen haben, so wurden die ersten beiden dort angehakten Clients vorausgewählt. Möchten Sie stattdessen andere Clients vergleichen, so wählen Sie diese aus der entsprechenden Liste aus und klicken Sie auf "Client nutzen". Möchten Sie eine zuvor gespeicherte Paketstatusdatei verwenden, so wählen Sie diese aus und laden Sie diese hoch. Ein Vergleich ist zwischen zwei Clients, zwei Paketstatusdateien oder einem Client und einer Paketstatusdatei möglich.

Zum Starten des Vergleiches klicken Sie abschließend auf "Paketstatus vergleichen".

Beim Vergleichen werden hierbei Paketname, Versionsnummer und Installationsstatus berücksichtigt. Diese Informationen werden, sofern sie vorliegen, jeweils in den Spalten "Paketname", "Version" und "Status" aufgelistet. In der Spalte "<=|>" sehen sie zudem, ob das Paket ausschließlich im linken ("<") oder rechten (">") Paketstatus vorhanden ist oder keine Übereinstimmung gefunden werden konnte ("|"). Sollte das Paket auf beiden Seiten vorhanden sein ("="), so werden zusätzlich die Versionsnummern und Installationszustände abgeglichen. Gefundene Unterschiede werden farblich hervorgehoben. Eine Zusammenfassung des Vergleiches sehen Sie am Ende der Tabelle.
