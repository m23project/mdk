CLIENT_desasterRecovery

Description: recover a client: all client jobs are done again, status is set to 0
Parameter:
  • clientName: name of the client
  • combineJobs: If set to true, the normal installation and removal jobs are combined to a m23normal and a m23normalRemove job otherwise the jobs are switched back to waiting status.
