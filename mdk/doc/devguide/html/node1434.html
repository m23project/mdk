CLCFG_genFstab

Description: generates the commands to auto detect the partitions and generate the fstab file
Parameter:
  • bootDevice: the device the bootloader should be installed on (e.g. /dev/hda)
  • rootDevice: the path to the installation partition (e.g. /dev/hda1)
